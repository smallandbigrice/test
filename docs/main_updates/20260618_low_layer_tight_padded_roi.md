- 帧差轮廓映射回原图后，四周外扩 32 像素。

- 将外扩后的原尺寸图像块居中贴入 640×640 画布。
- 画布其余区域统一填充值 114。
- 不进行特征融合，送入 YOLO 的仍是原始灰度相机图像块。
- 推理结果扣除画布偏移量后映射回原图。
- 检测框完全落在填充区域时直接丢弃。
- 超过 640×640 的运动区域回退为中心 640×640 裁剪。
